### 首先要注意几个概念
- 容器
  容器 可以理解为一个最简单的 linux 系统

- 镜像
  镜像 可以理解为容器里面需要安装部署的应用，这些应用只读不写

- 仓库
  仓库 类似 GitHub 的托管服务
