### 定义变量
1. 外部定义变量，使用 . 运算符运行 shell 文件

- 定义shell文件 `/tmp/test.sh`
``` shell
echo $aa
```

- 执行shell文件
``` bash
aa=123
. /tmp/test.sh # 这时会输出123
```

### 环境变量

  用户自定义变量只在当前的shell中生效，而环境变量会在当前shell和其所有子shell中生效。
  如果把环境变量写入相应的配置文件，那么这个环境变量就会在所有的shell中生效。

1. 导入环境变量
``` bash
source etc/profile
```

2. 内部定义环境变量
``` bash
export aa=/bin/
echo $aa
```
