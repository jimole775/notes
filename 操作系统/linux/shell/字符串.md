
# 单引号和双引号的区别
1. 单引号不能引用变量，不能转义单引号
``` shell
name='Rongxis'
say='my name is ${name}' # 直接输出 my name is ${name}

age='my \'age\' is 10' # 应该会直接报错
```

2. 双引号可以正常引用

``` shell
name='Rongxis'
name="my name is ${name}" # 输出 my name is Rpngxis

age="my \"age\" is 10" # 正确执行
```

3. 字段截取
``` shell
say="my name is Rongxis"
echo ${say:11:18} # 截取 11-18 的字符串
```

4. 字符串长度
``` shell
say="my name is Rongxis"
echo ${#say} # 输出say的长度
```
