- 最后总结一下：

在 nodejs 里面，使用 `__dirname` 生成的路径，结果都是 "\" 分隔符的，

如果在 webpack 中设置过 `__dirname` 的取值方式的，在 webpack 打包中，会把其中的 "\" 分隔符 转成 "/"

但是需要注意的是，业务模块中动态执行的 `__dirname` 还是原始的类型，所以还是需要手动转一下

> __dirname.spilt("\").join("/")
